Bienvenidos a Todos!

Como habran visto en las diferentes paginas que se tienen , se ha dado a conoser por medio de una galeria de imagenes lo que se utilizo para poder hacer el proyecto, claro esta que al principio no se tenia como tal previsto todo lo que se tiene ahora, pero si se seguira trabajando como se indico al principio con php


En nuestro repositorio git hub y en las respectivas entregas de proyecto que hemos hech, hemos estado avanzando poco a poco acerca del objetivo de la materia que se tiene,de poder aplicar la integracion continua a un proyecto que se va elaborando, con esto se quiere conseguir un metodo para la integracion de aportes que se puede dar un equipo de trabajo, podriamos decir que es un metodo agil para el desarrollo de un proyecto, claro esta que se establecen roles dentro de un equipo de trabajo, pero el objetivo es lograr aprender un metodo de trabajo agil


Tambien nosotros hemos podido ir viendo el como poder hacer las cosas y la integracion basandonos en la metodologia de investigacion, hacer una recoleccion de informacion suficiente para poder elaborar la actividad, tambien nos ha servido las conferencias que se tiene con el profesor


A lo largo del proceso hemos podido observar que la integracion continiua es muy importante, no solo por saber y aprenderlo sino que lo podemos aplicar a nuestro trabajo, en los casos de los informaticos, esto puede ser vital para poder optimizar procesos, en donde los particiapentes puedan realizar los aportes de forma continua y sin nunguna complicacion, en caso de otras carreras, puede llegar a servirte el conocimiento almenos para poder estructurar donde se puede compartir informacion por un medio accesible y que todos podamos trabajr sobre ello, lo que se quiere evitar con la integracion continua es que los trabajos no se entregen cada uno por una parte m como si el trabajo lo hubiesen hecho diferentes personas con diferente estilo


Se puede llegar tener mas informacion acerca del proyecto viendo las entegas que hemos hecho desde la primera ,asi se detalla todo el proceso que hemos hecho para realizar el proyecto
